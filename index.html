Exposición: Servidores Web

¿Qué es un servidor web?

Un servidor web es un sistema que almacena, procesa y entrega páginas web a los usuarios. Funciona utilizando el protocolo HTTP o HTTPS, y responde a las peticiones que hacen los navegadores cuando visitamos un sitio web.

Tipos de servidores web

1. Servidores HTTP

Son los más comunes. Algunos ejemplos son Apache, Nginx y LiteSpeed. Entregan contenido como HTML, imágenes y videos.

2. Servidores de aplicaciones

Ejecutan aplicaciones web complejas. Ejemplos: Tomcat (Java), Node.js (JavaScript), .NET Core (C#).

3. Servidores de archivos

Usados para compartir archivos en red. Ejemplos: FTP, Samba, NFS.

4. Servidores en la nube

Funcionan desde plataformas como AWS, Google Cloud o Azure. Son escalables y flexibles.

5. Servidores proxy y caché

Actúan como intermediarios y mejoran el rendimiento. Ejemplos: Varnish, Squid.

Pasos para subir una página web

  1. Crear el archivo HTML y los recursos (imágenes, CSS, etc.).
  2. Elegir una plataforma para subirla (como Netlify, GitHub Pages o Vercel).
  3. Subir los archivos a esa plataforma.
  4. Obtener un enlace para compartir tu sitio en Internet.
